A mutable priority queue implementation used primarily by Dijkstra\textquotesingle{}s algorithm. 

This data structure maintains elements in a binary heap, allowing efficient extraction of the minimum element and updating priorities of existing elements. The class T must have\+: (i) accessible field int queue\+Index (ii) operator\texorpdfstring{$<$}{<} defined for comparison

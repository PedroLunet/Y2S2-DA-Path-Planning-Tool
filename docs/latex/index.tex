\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Project Overview}{Project Overview}}\label{index_autotoc_md1}
A comprehensive transportation routing system implementing multiple routing algorithms with a focus on environmental friendliness. This application allows users to find optimal routes between locations using various transport modes, including a specialized "{}driving-\/walking"{} mode for eco-\/friendly routing.\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Standard \doxylink{class_routing}{Routing}}}\+: Find the fastest route between two locations using Dijkstra\textquotesingle{}s algorithm
\item {\bfseries{Restricted \doxylink{class_routing}{Routing}}}\+: Calculate routes that avoid specific nodes or segments
\item {\bfseries{Environmentally-\/\+Friendly \doxylink{class_routing}{Routing}}}\+: Find optimal routes that combine driving and walking to reduce emissions
\item {\bfseries{Interactive \doxylink{class_menu}{Menu} Interface}}\+: Easy-\/to-\/use terminal interface for accessing all functionality
\item {\bfseries{File Input/\+Output Support}}\+: Process routing requests from files and save results
\end{DoxyItemize}\hypertarget{index_autotoc_md3}{}\doxysection{\texorpdfstring{Core Components}{Core Components}}\label{index_autotoc_md3}

\begin{DoxyItemize}
\item {\bfseries{\doxylink{class_graph}{Graph} Structure}}\+: Template-\/based graph implementation with vertices and edges
\item {\bfseries{\doxylink{class_routing}{Routing} Algorithms}}\+: Implementations of Dijkstra\textquotesingle{}s algorithm and specialized routing functions
\item {\bfseries{\doxylink{class_menu}{Menu} System}}\+: User interface handling for all routing operations
\item {\bfseries{Parser}}\+: Data loading and processing from various file formats
\item {\bfseries{\doxylink{class_graph}{Graph} Builder}}\+: Construction of the routing graph from location and distance data
\end{DoxyItemize}\hypertarget{index_autotoc_md4}{}\doxysection{\texorpdfstring{Time Complexity}{Time Complexity}}\label{index_autotoc_md4}

\begin{DoxyItemize}
\item {\bfseries{Dijkstra\textquotesingle{}s Algorithm}}\+: O(\+E log V) where E is the number of edges and V is the number of vertices
\item {\bfseries{Environmentally-\/\+Friendly \doxylink{class_routing}{Routing}}}\+: O(\+P \texorpdfstring{$\ast$}{*} E log V) where P is the number of potential parking locations
\item {\bfseries{\doxylink{class_graph}{Graph} Building}}\+: O(V + E) for constructing the graph from data
\end{DoxyItemize}\hypertarget{index_autotoc_md5}{}\doxysection{\texorpdfstring{Usage}{Usage}}\label{index_autotoc_md5}

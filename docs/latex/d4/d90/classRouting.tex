\doxysubsection{Detailed Description}
Provides algorithms and utilities for route planning and path finding. 

This class implements various routing algorithms including Dijkstra\textquotesingle{}s shortest path, alternative route finding, and environmentally-\/friendly routes that combine driving and walking segments.

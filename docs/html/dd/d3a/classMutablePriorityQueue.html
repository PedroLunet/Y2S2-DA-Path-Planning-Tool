Detailed Description

template<class T>
class MutablePriorityQueue< T >

A mutable priority queue implementation used primarily by Dijkstra's algorithm.

This data structure maintains elements in a binary heap, allowing efficient extraction of the minimum element and updating priorities of existing elements. The class T must have: (i) accessible field int queueIndex (ii) operator< defined for comparison

TThe type of elements stored in the queue
